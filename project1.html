A quick overview of the bootstrap grid system

  • As you might expect there are rows and columns. All you have to do is assign the class row or col to a div (this also works on most other items too).
  • 12 is the max number of columns allowed in a row. Any after that will fall on a new line.
  • You can add a break div - and it can be responsive. See your gallery code :)
  • You can make them responsive by specifying sizes - sm, md, lg which correspond to screen sizes and the media queries they provide. The default col with no sizing specified is xs.
  • You can also offset cols to shift them along. I've used this extensively here.
  • You can also adjust the order of the columns with numbers and first/last classes. It's a pretty powerful grid system.

I'll give an example of the grid system written inside a blockquote so you can see the style. col-sm-6 means from small screens and upwards the column will be 6 or half the size of the page (or div if it's nested). It will change to a col-12 automatically for xs screen sizes.
